# 连环画小程序
## 微信小程序原生开发
## 技术栈
1. gulp项目构建
2. vant ui
3. less样式

## 开发调式
1. 安装依赖
```
npm install
```
2. 开发模式
```
npm run dev
```
3. 开发带mock api模式
```
npm run mock
```
4. 生产模式
```
npm run build
```
